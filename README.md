Las cookies en JavaScript son la parte fundamental que se utilizan para leer y crear los códigos desde 0 por ejemplo para crear el dato de una persona, pone ponerle el modo idioma y edad un es un ejemplo muy sencillo sería por ejemplo; "usuario=milton; Id=17; idioma=español"

*AREAS DE SECCIONESEN JAVASCRIPT*

Bueno para empezar las áreas de  secciones en JavaScript  son como los ámbitos que definen dónde una variable se vaya a disponir para usarla dentro de algún código. Por ejemplo cada vez que crean una sección ya sea con una fusión o un bloque o directamente en el archivo completo estará limitando un espacio con reglas propia.

Entonces lo que se estaría declarando dentro de esas áreas solo pueden leerse y modificarse ahí y esto no afectaría  en ninguna otras partes del programa elaborado,  asi sí se evitarían conflictos cuando por ejemplo si  2 nombres de diferentes usuario sean iguales  se mantendrían sus variables separadas no habría ningún problema de confusión en ello.

En pocas  palabras estos ámbitos funcionan como límites invisibles que se organizan en los  código controlando las visibilidades de los datos y hacen que el programa funcione y sea más predecible y menos propenso a errores y posible riesgo de virus.

*ELIMINAR SECCIONES EN JAVASCRIPT*

Bueno, por la eliminación de secciones en llaves crep, sería, por ejemplo, eliminar toda generalmente eliminar todos los datos en alguna página web elaborada algunos ejemplos de los de las eliminaciones serían el DIV,  SECTION y  ARTICLE eso se utilizaría en las manipulaciones del don que sería documentos del modelo.

Uno de los  método más moderno es el REMOVE () esta es la forma más sencilla y moderna para eliminar un elemento básico directamente de la página creada.

El el otro método sería por el REMOVECHILD () 
Ese es uno de los métodos más básicos y tradicionales para usarlo se necesitaría seleccionar primero los elementos como por ejemplo el elemento padre y luego indicar cuál de cuál sería el elemento los siguientes elementos por ejemplo los elementos hijos que queremos quitar de la página web.
